Description

Let's say that you are reading some very long epic story about heroes. At some point you wonder (for some reason) which hero name appears the most in a story. You write down the names inside this app, you pick the story to be scanned, and boom... Name "Frodo" appeared the most, i know, i wanted "Legolas" or "Gimli" too. That is what this app does in a nutshell.
There is much more, but about that in a challenge.

Challenge

I mentioned that app can do much more, so let's go back to our heroes example for a second. We know that every story with heroes must have a bad guys. Beside just writing their names and scan for how many times their names shows up in a story, we can also value those names with any number. We can say that Legolas is a positive hero, so we can value him for example with number 1. Then we can value Saruman with -1, since he is a bad one. If we scan our story now, we will get +1 for every time that name Legolas appears in a story, and -1 for Saruman. Then a total score that shows final number, positive or negative, depending on witch name appeared more in a story. We can also add number 0, but then that hero is neutral.

I'm talking about heroes because that is how i like to explain. You can use this app to scan text from emails. You can pick the words you love to see in you emails, and with one click, you will know which email contains that word.

I see challenge as a title, but i see no challenge. Well, today it's not, we have now "includes" in JavaScript. We can just break down text into list(array) of words and just check them with "includes". But this project was made before that. To scan the words, i couldn't just pick the text and check if my word is inside that text. So i breaked down the text into huge list of words. Thats great, but that was still far from goal. If Comparing "Legolas" with "Legolas.", those are two different words, since second one has a dot at the end. So i trimed empty spaces and dots, special chars etc. At this point i had it. I just needed to compare the words in lowecase or uppercase. That was a small challenge here.
